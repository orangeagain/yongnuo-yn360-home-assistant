# Yongnuo BLE Protocol Reference

永诺 YN360 / YN150 系列灯具 BLE 控制协议文档。

## 适用型号

| BLE 广播名 | 型号 | RGB | 色温 | 色温通道 (CH) | 验证状态 |
|------------|------|-----|------|--------------|----------|
| `YUNM...` / `Yongnuo LED` | YN360 系列 | ✅ | ✅ | `0x01` | 已验证 |
| `YN150Ultra RGB` | YN150 Ultra RGB | ✅ | ✅ | `0x00` | 已验证 |
| `YN150WY` | YN150 WY | ❌ 无RGB LED | ✅ | `0x00` | 已验证 |

## GATT 服务与特征

所有型号共用同一组 GATT 服务。

### 主服务 `f000aa60-0451-4000-b000-000000000000`

| UUID | 属性 | 用途 |
|------|------|------|
| `f000aa61-0451-4000-b000-000000000000` | write | **命令通道**（发送控制指令） |
| `f000aa63-0451-4000-b000-000000000000` | notify | 状态反馈 |
| `0000fff3-0000-1000-8000-00805f9b34fb` | write | 附加写入通道（用途未知） |
| `0000fff4-0000-1000-8000-00805f9b34fb` | notify, write | 双向通道（用途未知） |
| `0000fff5-0000-1000-8000-00805f9b34fb` | read, write | 返回 `CHAR5_VALUE`（占位符） |

## 数据包格式

所有命令均为 **6 字节定长包**，大端序。

```
字节位置:  [0]     [1]     [2]     [3]     [4]     [5]
通用含义:  包头    命令码  参数1   参数2   参数3   包尾
固定值:    0xAE                                    0x56
```

| 字节 | 位置 | 说明 |
|------|------|------|
| `[0]` 包头 | 固定 `0xAE` | 所有指令必须以此开头，否则设备忽略 |
| `[1]` 命令码 | `0xA1` / `0xAA` / `0xA3` | 决定指令类型，见下方命令列表 |
| `[2]` 参数1 | 取决于命令码 | RGB 指令=红色值；色温指令=通道字节；关灯=0x00 |
| `[3]` 参数2 | 取决于命令码 | RGB 指令=绿色值；色温指令=冷白亮度；关灯=0x00 |
| `[4]` 参数3 | 取决于命令码 | RGB 指令=蓝色值；色温指令=暖白亮度；关灯=0x00 |
| `[5]` 包尾 | 固定 `0x56` | 所有指令必须以此结尾 |

## 命令列表

### 1. RGB 颜色控制 — 命令码 `0xA1`

设置 RGB 颜色，同时开灯。

```
字节:   [0]     [1]     [2]     [3]     [4]     [5]
含义:   包头    命令码  红(R)   绿(G)   蓝(B)   包尾
值:     0xAE    0xA1    0x00-FF 0x00-FF 0x00-FF 0x56
```

| 字节 | 位置 | 含义 | 范围 | 说明 |
|------|------|------|------|------|
| `[0]` | 包头 | 固定 | `0xAE` | |
| `[1]` | 命令码 | RGB 模式 | `0xA1` | |
| `[2]` | 红色 (R) | 红色通道亮度 | `0x00` - `0xFF` (0-255) | 0=灭，255=最亮 |
| `[3]` | 绿色 (G) | 绿色通道亮度 | `0x00` - `0xFF` (0-255) | 0=灭，255=最亮 |
| `[4]` | 蓝色 (B) | 蓝色通道亮度 | `0x00` - `0xFF` (0-255) | 0=灭，255=最亮 |
| `[5]` | 包尾 | 固定 | `0x56` | |

**示例：**

| 指令 | R | G | B | 颜色 |
|------|---|---|---|------|
| `AE A1 FF 00 00 56` | 255 | 0 | 0 | 红色 |
| `AE A1 00 FF 00 56` | 0 | 255 | 0 | 绿色 |
| `AE A1 00 00 FF 56` | 0 | 0 | 255 | 蓝色 |
| `AE A1 FF FF FF 56` | 255 | 255 | 255 | 白色 最大亮度 |
| `AE A1 FF 80 00 56` | 255 | 128 | 0 | 橙色 |
| `AE A1 FF FF 00 56` | 255 | 255 | 0 | 黄色 |
| `AE A1 00 FF FF 56` | 0 | 255 | 255 | 青色 |
| `AE A1 FF 00 FF 56` | 255 | 0 | 255 | 品红 |
| `AE A1 80 00 FF 56` | 128 | 0 | 255 | 紫色 |
| `AE A1 11 11 11 56` | 17 | 17 | 17 | 暗白色 (低亮度) |

**适用型号：** YN360、YN150Ultra RGB

> **注意：** YN150WY 会响应 A1 指令（开灯），但 RGB 值被忽略，因为硬件没有 RGB LED。

### 2. 色温控制 — 命令码 `0xAA`

设置冷白 / 暖白亮度，控制色温。

```
字节:   [0]     [1]     [2]     [3]     [4]     [5]
含义:   包头    命令码  通道(CH) 冷白(CW) 暖白(WW) 包尾
值:     0xAE    0xAA    0x00/01 0x00-63  0x00-63  0x56
```

| 字节 | 位置 | 含义 | 范围 | 说明 |
|------|------|------|------|------|
| `[0]` | 包头 | 固定 | `0xAE` | |
| `[1]` | 命令码 | 色温模式 | `0xAA` | |
| `[2]` | 通道 (CH) | 设备系列标识 | `0x00` 或 `0x01` | YN150=0x00，YN360=0x01，发错通道无响应 |
| `[3]` | 冷白 (CW) | 冷白 LED 亮度 (≈5500K) | `0x00` - `0x63` (0-99) | **最大值 99，不是 255** |
| `[4]` | 暖白 (WW) | 暖白 LED 亮度 (≈3200K) | `0x00` - `0x63` (0-99) | **最大值 99，不是 255** |
| `[5]` | 包尾 | 固定 | `0x56` | |

> **重要：** CW/WW 范围是 0-99（`0x00`-`0x63`），不是 0-255。超过 99 的值会被设备忽略。

**通道字节 `[2]` (CH) 对照表：**

| 值 | 适用型号 | 说明 |
|----|----------|------|
| `0x00` | YN150 系列 | YN150Ultra RGB、YN150WY 均使用；发送 0x01 无响应 |
| `0x01` | YN360 系列 | YN360 使用；发送 0x00 无响应 |

**示例（YN150 系列，CH=0x00）：**

| 指令 | CW | WW | 效果 |
|------|----|----|------|
| `AE AA 00 63 00 56` | 99 | 0 | 冷白 最大亮度 |
| `AE AA 00 00 63 56` | 0 | 99 | 暖白 最大亮度 |
| `AE AA 00 32 32 56` | 50 | 50 | 混合 中等亮度 |
| `AE AA 00 63 63 56` | 99 | 99 | 混合 最大亮度 |
| `AE AA 00 00 0A 56` | 0 | 10 | 暖白 低亮度 |
| `AE AA 00 0A 00 56` | 10 | 0 | 冷白 低亮度 |

**示例（YN360，CH=0x01）：**

| 指令 | CW | WW | 效果 |
|------|----|----|------|
| `AE AA 01 63 00 56` | 99 | 0 | 冷白 最大亮度 |
| `AE AA 01 00 63 56` | 0 | 99 | 暖白 最大亮度 |
| `AE AA 01 32 32 56` | 50 | 50 | 混合 中等亮度 |

### 3. 关灯 — 命令码 `0xA3`

关闭灯具。

```
字节:   [0]     [1]     [2]     [3]     [4]     [5]
含义:   包头    命令码  保留    保留    保留    包尾
值:     0xAE    0xA3    0x00    0x00    0x00    0x56
```

| 字节 | 位置 | 含义 | 值 | 说明 |
|------|------|------|-----|------|
| `[0]` | 包头 | 固定 | `0xAE` | |
| `[1]` | 命令码 | 关灯 | `0xA3` | |
| `[2]` | 保留 | 无意义 | `0x00` | 固定填 0 |
| `[3]` | 保留 | 无意义 | `0x00` | 固定填 0 |
| `[4]` | 保留 | 无意义 | `0x00` | 固定填 0 |
| `[5]` | 包尾 | 固定 | `0x56` | |

**适用型号：** 全部

## 完整指令速查表

```
所有指令格式:  AE [命令码] [参数1] [参数2] [参数3] 56
```

**RGB 颜色 (命令码=A1)：**

| 指令 | 颜色 |
|------|------|
| `AE A1 FF 00 00 56` | 红色 |
| `AE A1 00 FF 00 56` | 绿色 |
| `AE A1 00 00 FF 56` | 蓝色 |
| `AE A1 FF FF FF 56` | 白色 (最大) |
| `AE A1 FF FF 00 56` | 黄色 |
| `AE A1 00 FF FF 56` | 青色 |
| `AE A1 FF 00 FF 56` | 品红 |
| `AE A1 FF 80 00 56` | 橙色 |
| `AE A1 80 00 FF 56` | 紫色 |

**色温 (命令码=AA)：**

| 指令 | 型号 | 效果 |
|------|------|------|
| `AE AA 00 63 00 56` | YN150 | 冷白 100% |
| `AE AA 00 00 63 56` | YN150 | 暖白 100% |
| `AE AA 00 32 32 56` | YN150 | 混合 50/50 |
| `AE AA 01 63 00 56` | YN360 | 冷白 100% |
| `AE AA 01 00 63 56` | YN360 | 暖白 100% |
| `AE AA 01 32 32 56` | YN360 | 混合 50/50 |

**关灯 (命令码=A3)：**

| 指令 | 说明 |
|------|------|
| `AE A3 00 00 00 56` | 全型号通用 |

## 调试工具

项目附带 `debug_ble.py`，可独立运行（不依赖 Home Assistant）：

```bash
# 扫描附近 BLE 设备
python debug_ble.py scan

# 查看 GATT 服务
python debug_ble.py services <地址>

# 监听通知
python debug_ble.py sniff <地址>

# 手写指令
python debug_ble.py write <地址> f000aa61-0451-4000-b000-000000000000 AEA1FF000056

# 综合自动测试（先色温后 RGB，一键全测）
python debug_ble.py auto-test <地址>

# 自动 RGB 测试（红/绿/蓝渐变、色相环、白色亮度）
python debug_ble.py auto-rgb <地址>

# 自动色温测试（冷白/暖白渐变、双通道同步、交叉渐变，测试 channel 0x00 和 0x01）
python debug_ble.py auto-ct <地址>
```
